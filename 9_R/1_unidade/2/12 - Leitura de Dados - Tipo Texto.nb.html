Leitura de Arquivos de Texto no R

Leitura de Arquivos de Texto no R

Nesta aula, vamos explorar como trabalhar com arquivos estruturados no formato de texto simples (plain text) no R. O R uma ferramenta poderosa para anlise de dados, e esses dados podem estar disponveis em diferentes tipos de formatos, que se dividem em trs categorias principais:

Tipos de Dados

Tipos de Dados

1. Dados Estruturados ou Tabulares

Estes so os dados mais comuns em anlise de dados. Eles esto organizados em formato de tabela, onde: - Linhas correspondem s observaes. - Colunas correspondem s variveis.

Esses dados geralmente so disponibilizados como: - Arquivos de texto simples (CSV, TSV, etc.): Arquivos que podem ser abertos em editores de texto como o Bloco de Notas ou o RStudio. - Planilhas eletrnicas: Tabelas armazenadas em arquivos binrios (como Excel). - Tabelas em bancos de dados relacionais: Comumente acessadas por meio de SQL.

2. Dados Semi-estruturados

Esses dados tm uma estrutura definida, mas no so organizados em tabelas simples: - Arquivos XML: Usados em APIs Web, HTML, KML (informao geogrfica), etc. - Arquivos JSON: Estruturas hierrquicas de armazenamento de dados.

3. Dados No Estruturados

Dados que no seguem uma estrutura tabular clara, como: - Imagens - udios - Vdeos

Mesmo que esses arquivos tenham uma estrutura interna (como pixels para imagens ou amostras para udio), a informao relevante, como a presena de um objeto em uma imagem, no explicitamente organizada.

Leitura de Arquivos CSV

O Formato CSV

O formato CSV (Comma Separated Values) o mais comum para armazenar dados tabulares. Os valores so separados por um delimitador (geralmente uma vrgula ,). Em muitos pases de lngua portuguesa, o delimitador ; devido ao uso de , como separador decimal.

Explicao dos Parmetros

  • header: Indica se o arquivo contm um cabealho (nomes das colunas).
  • sep: Define o delimitador que separa os campos (, para CSV e ; para CSV no padro portugus).
  • dec: Define o separador decimal (. ou ,).
  • quote: Define como so tratadas as aspas (til para strings que contm o separador).

Leitura de Arquivos TSV

O Formato TSV

TSV (Tab Separated Values) semelhante ao CSV, mas os valores so separados por tabulao (\t) em vez de vrgulas. Isso til para evitar ambiguidades com vrgulas dentro dos dados.

Exemplo de Arquivo TSV

Lendo Arquivos TSV no R

A leitura de arquivos TSV muito semelhante leitura de arquivos CSV, mas alteramos o parmetro sep para \t.

# Usando read.table para ler um arquivo TSV

Nesta aula, vamos explorar como podemos coletar dados diretamente do usurio dentro do R, permitindo que o prprio usurio insira informaes no terminal. Essa tcnica til para criar formulrios interativos ou para capturar entradas especficas durante a execuo do cdigo.

Introduo Funo scan()

A funo scan() permite a entrada de dados diretamente pelo usurio no console. Diferente da leitura de dados de arquivos, a funo scan() usada para capturar a entrada do usurio em tempo real. Isso til para situaes em que precisamos que o usurio fornea informaes especficas, como nome, idade, ou qualquer outra entrada.

Estrutura Bsica da Funo

A funo scan() pode ser combinada com outras funes, como cat(), para criar interaes simples e intuitivas. A funo cat() usada para exibir uma mensagem ou pergunta ao usurio, enquanto a funo scan() captura a resposta.

Exemplo Alterado: Coletando Informaes Pessoais

Vamos criar uma funo que solicita o nome, a altura e o peso do usurio, e ento calcula o ndice de Massa Corporal (IMC) com base nas informaes fornecidas. O IMC uma medida que pode ser usada para avaliar se uma pessoa est dentro de um peso saudvel para sua altura.


calcular_imc <- function() {
    cat("Qual o seu nome?",
        "[Aps escrever, pressione ENTER]\n")
    nome <- scan(what = "character", nmax = 1, quiet = TRUE)
    
    cat("Qual  a sua altura em metros?",
        "[Por exemplo, 1.75]\n")
    altura <- as.numeric(scan(what = "numeric", nmax = 1, quiet = TRUE))
    
    cat("Qual  o seu peso em quilogramas?",
        "[Por exemplo, 68.5]\n")
    peso <- as.numeric(scan(what = "numeric", nmax = 1, quiet = TRUE))
    
    # Verificao para garantir que os valores sejam numricos
    if (is.na(altura) || is.na(peso)) {
        cat("Erro: Certifique-se de que a altura e o peso so nmeros vlidos.\n")
        return(NULL)
    }
    
    imc <- peso / (altura^2)
    
    cat("Seja bem-vindo(a), ", nome, ".\n", sep = "")
    cat("Seu IMC  ", round(imc, 2), ".\n", sep = "")
    
    classificacao <- ifelse(imc < 18.5, "Abaixo do peso",
                   ifelse(imc >= 18.5 & imc <= 24.9, "Peso normal",
                   ifelse(imc >= 25 & imc <= 29.9, "Sobrepeso",
                          "Obesidade")))
    
    cat("Com base no seu IMC, voc est na categoria: ", classificacao, ".\n", sep = "")
}

calcular_imc()

Explicao do Cdigo

  • cat(): Exibe mensagens ao usurio. usado aqui para fazer perguntas e fornecer informaes.
  • scan(): Captura a entrada do usurio. Aqui, usamos what = "character" para capturar o nome como texto e what = "numeric" para capturar altura e peso como nmeros.
  • Clculo do IMC: O IMC calculado dividindo o peso pela altura ao quadrado (peso / (altura^2)).
  • Classificao do IMC: Aps o clculo do IMC, utilizamos uma srie de ifelse para classificar o resultado em categorias (Abaixo do peso, Peso normal, Sobrepeso, Obesidade).
